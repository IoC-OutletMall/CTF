# Password Cracking CTF

A beginner-friendly Capture the Flag challenge focused on password cracking fundamentals. This CTF provides an engaging, educational experience for those looking to develop their password cracking skills from the ground up.

## Purpose

This repository hosts a series of password cracking challenges designed to be both fun and educational. Whether you're completely new to password cracking or have some experience, these challenges will help you build practical skills in hash analysis, cracking techniques, and security tool usage.

Each challenge includes hints to guide you if you get stuck, making this an ideal learning environment for developing cybersecurity skills at your own pace.

## What You'll Learn

- **Hash Type Identification**: Learn to recognize different hash types based on their format and characteristics
- **Password Cracking Tools**: Gain hands-on experience with industry-standard tools like Hashcat and John the Ripper
- **Wordlist Usage**: Understand how to effectively use wordlists for dictionary attacks
- **Brute Force Techniques**: Learn to generate and apply password masks for targeted brute-force attacks
- **Cracking Strategies**: Develop intuition for which techniques work best for different scenarios

## Target Audience

This CTF is designed for:
- Complete beginners with no prior password cracking experience
- Security enthusiasts looking to understand authentication weaknesses
- Students learning cybersecurity fundamentals
- Anyone wanting hands-on practice with offensive security tools

## Challenge Structure

The CTF consists of **10 progressive challenges** that gradually increase in difficulty, covering:

- Basic hash identification
- Simple dictionary attacks
- Combination attacks
- Rule-based cracking
- Mask-based brute forcing
- Advanced cracking techniques

## Tools You'll Use

- **Hashcat**: GPU-accelerated password cracking
- **John the Ripper**: Versatile password cracking utility
- **Wordlists**: Common password dictionaries. Can be obtained from <a href="https://example.com"><code>Kali</code></a>

- **Hash analyzers**: Tools for identifying hash types
- **Custom masks**: Self-generated patterns for targeted attacks

## Getting Started

1. Clone this repository
2. Navigate to the "Crack the Hash" directory
3. Start with Challenge 1 and work your way through
4. Use the hints if you need guidance
5. Have fun and learn!
